修bug日记#1

· 2024-08-08 · # 网站更新 # 日常

怎么说呢？网站在昨天出了点小小的bug（虽然一个网站菜鸡还是用了半天才解决），今天大家请欣赏我的修bug迷惑行为吧🤣
事情是这样的：昨天有人向我要微博，然后可恶的我妈竟然说要的是博客......然后我就发现GitHub，额，它一定要我双重验证！！！

好的，GitHub 2FA（即为双重验证简称）有两种选择：
1）发个消息到你的手机号，但不支持中国大陆
2）下载一个国外App，并扫码登录
这样可难住了我......
理由如下：
1）我没有国外手机号
2）我没有手机
3）我不能下载国外App

这样，GitHub这条路就彻底被封了......（其实还没有，只是我后知后觉）

好的，那我把源码转到Gitee上去，这样总可以了吧......
我很快就完成了这个任务，但是出来的页面却是：
Bad Request （错误的请求）
403 Forbidden Unknown Hostname （未知的域名）

这可怎么办呢？今天一早，我又查了一下Gitee如何建网站，发现需要Gitee Pages服务——但当我打开Gitee Pages的时候，却发现服务已经无法使用了

这可怎么办哪？于是我又转战GitHub，试图通过2FA......
结果你可能都知道了：在B站上一搜，发现方法就近在眼前啊！
原来装个插件就好了......

这下，bug终于修完了，可以畅快地访问了！😁
